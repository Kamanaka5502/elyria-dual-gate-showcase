# Dual-Gate Model

The Elyria Systems dual-gate structure separates meaning from consequence.

## Gate A — Semantic Integrity

Evaluates:

- structural coherence
- completeness of meaning
- ambiguity
- contradiction
- readiness for handoff

Outcome:

- PASS → may proceed to Gate B
- FAIL → REFUSE or REDIRECT

## Gate B — Execution Boundary

Evaluates:

- authority
- admissibility
- state
- constraint
- capacity
- policy
- commit boundary

Outcome:

- EXECUTE
- REFUSE
- ESCALATE
- REDIRECT
- HALT

## Core rule

A transition may bind only if both gates pass.

If Gate A fails → the system does not produce a valid candidate.

If Gate B fails → the system does not allow consequence to bind.

## Public distinction

Most systems combine these layers.

This model separates them explicitly.
